# 🔐 Retail Security Risk Analysis

This project presents a real-world cybersecurity risk analysis conducted in a retail environment.

## 📍 Scenario

During a system review, a critical vulnerability was identified:

- Exposure of stored credentials within the system
- Access to a master account with visibility across multiple locations

## 🚨 Risk Identified

Unauthorized access to CCTV systems due to insecure credential handling.

## 📊 Risk Level

🔴 Critical

## 🧠 Methodologies Used

- Bow Tie Analysis
- SWOT Analysis
- Root Cause Analysis
- Risk Assessment
- PDCA Cycle

## 🛡️ Key Findings

- Credentials stored in a visible/recoverable format
- Lack of access control
- Potential access to multiple environments (stores, office, factory)

## 💡 Insight

Critical vulnerabilities often arise from simple security misconfigurations rather than advanced attacks.
